   Properties:
   - FirebaseAuth _auth

   - RxString verificationId

   Methods:
   - onReady() → Initialize auth state listener
   - _setInitialScreen(User?) → Navigate based on auth state
   - createUserWithEmailAndPassword(email, password) → Future<void>
   - loginWithEmailAndPassword(email, password) → Future<void>
   - phoneAuthentication(phoneNo) → Future<void>
   - verifyOTP(otp) → Future<bool>
   - logout() → Future<void>

   Features:
   - Auto-redirect to Dashboard if logged in
   - Auto-redirect to Welcome if logged out
   - Error handling with GetX snackbars
   - Firebase exception handling

2. UserRepository (Singleton with GetX):

   Properties:
   - FirebaseFirestore _db

   Methods:
   - createUser(UserModel) → Future<void>
   - getUserDetails(email) → Future<UserModel>
   - updateUser(UserModel) → Future<void>

   Features:
